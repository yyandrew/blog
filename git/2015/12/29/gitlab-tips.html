gitlab tips

Dec 29, 2015

  • 利用Omnibus package installation只需2分钟可以安装完成
  • gitlab配置文件默认在/etc/gitlab/gitlab.rb和/var/opt/gitlab/gitlab-rails/etc/gitlab.yml
  • 进入production数据库sudo -u gitlab-psql psql -h /var/opt/gitlab/postgresql -d gitlabhq_production
  • 手动激活一个新用户gitlabhq_production=# update users set confirmed_at=NOW() where email='example@example.com';
  • 禁止gitlab开机自动启动systemctl disable gitlab-runsvdir.service
