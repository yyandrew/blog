使用webpack和babel配置ReactJS开发环境

Sep 24, 2018

什么是ReactJS

它是由Facebook开源的用来构建用户界面的JavaScript库。它采用虚拟DOM机制向DOM填充数据。更新单独的虚拟DOM组件比更新整个DOM树要快。

什么是Webpack

它是一个模块打包工具。它的主要目标是将JavaScript文件打包在一起之后在浏览器中使用。同时它还可以用来转换(transform)。

什么是Babel

语法转换工具。将最新的JavaScript语法转换成浏览器支持的语法，无需等到浏览器支持这些最新的语法。

创建一个ReactJS项目

一、 创建一个新的项目目录，及基本的文件。

mkdir reactjs-demo
cd reactjs-demo
npm init -y
mkdir dist
cd dist

二、 新建一个dist/index.html文件

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ReactJS示例</title>
  </head>

  <body>
    <div id="app"></div>
    <script src="/bundle.js"></script>
  </body>
</html>

其中bundle.js是由Webpack产生的一个打包文件。

此时项目的目录结构为:


├── dist
│   └── index.html
└── package.json

配置Webpack

一、安装Webpack

npm install --save-dev webpack webpack-dev-server webpack-cli

二、修改package.json

...
"scripts": {
  "start": "webpack-dev-server --config ./webpack.config.js --mode development",
  ...
},
...

三、新建webpack.config.js文件

module.exports = {
  entry: './src/index.js',
  output: {
    path: __dirname + '/dist',
    publicPath: '/',
    filename: 'bundle.js'
  },
  devServer: {
    contentBase: './dist'
  }
}

其中entry是入口起点，它指示webpack应该使用哪个模块，来作为构建其内部依赖图的开始。output告诉webpack在哪里输出它所创建的打包文件。devServer指定dist文件夹为资源文件夹。

四、创建src/index.js文件

mkdir src
cd src
touch index.js

src/index.js的内容

console.log('ReactJS示例');

此时的目录结构为：

├── dist
│   └── index.html
├── node_modules
├── package.json
├── package-lock.json
├── src
│   └── index.js
└── webpack.config.js

五、启动项目 sh npm start

六、在浏览器中打开http://localhost:8080，不出意外此时在浏览器的console里会打印出ReactJS示例。

配置Babel

一、安装依赖包

npm install --save-dev @babel/core @babel/preset-env babel-loader @babel/preset-react

二、修改文件webpack.config.js内容

...
entry: './src/index.js',
module: {
  rules: [
    {
      test: /\.(js|jsx)$/,
      exclude: /node_modules/,
      use: ['babel-loader']
    }
  ]
},
resolve: {
  extensions: ['*', '.js', '.jsx']
},
...

三、在项目根目录下创建新文件.babelrc

touch .babelrc

.babelrc文件内容如下：

{
  "presets": [
    "@babel/preset-env",
    "@babel/preset-react"
  ]
}

使用ReactJS

一、在项目根目录下安装如下依赖

npm install --save react react-dom

二、修改webpack.config.js配置自动刷新浏览器

var webpack = require('webpack')
...
plugins: [
  new webpack.HotModuleReplacementPlugin()
],
devServer: {
  ...
  hot: true
}
...

三、修改文件src/index.js内容使用ReactJS

import React from 'react';
import ReactDOM from 'react-dom';

const title = 'ReactJS示例';

ReactDOM.render(
  <div>{title}</div>,
  document.getElementById('app')
)
